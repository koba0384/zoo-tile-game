
# どうぶつえんタイルゲーム試作版

Streamlit で動く、カルカソンヌ風の **動物園タイル配置ゲーム**の雛形です。  
まずは **デジタルで遊んで調整**してから、あとでアナログ版に落とし込めるようにしてあります。

## いま入っているもの

- 2〜4人のローカル対戦
- 山札からタイルを引く
- タイルの回転
- 柵あり / 柵なしの辺一致チェック
- 囲いの接続判定
- ミープル配置
- 多数決得点
- 同点トップは全員満点
- ライオンの単独飼育ルール
- 仲良しポイント
- サバンナポイント
- にぎやかポイント
- 群れポイント
- 完成ボーナス（囲いのタイル数）
- 内包ボーナス（完成済みの内側の囲いの完成ボーナス）
- 山札切れ時の未完成囲いの動物点精算
- 置けないタイルを山札の下に送って引き直し

```bash

```

## Streamlit Community Cloud に上げる流れ

1. このフォルダを GitHub に push
2. Streamlit Community Cloud でリポジトリを選ぶ
3. `app.py` を指定して deploy

## ルールの実装メモ

### 囲い
- タイルの `open` 辺同士でつながっているマスのまとまりを1つの囲いとして扱います。
- `fence` 辺は仕切りです。

### ミープル
- 置いたタイルが属する囲いに **その時点でミープルが1個もなければ** 置けます。
- あとから別囲い同士がつながって、複数人のミープルが同じ囲いに入るのは OK です。

### 得点
- 完成した囲いは多数決
- 同点トップは全員満点
- 未完成の囲いは終了時に **動物点のみ**
- コンボ点と完成ボーナスは **完成時だけ**
